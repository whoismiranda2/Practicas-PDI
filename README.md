# Prácticas PDI

Recurso realizado para guía en la realización de prácticas, el uso del código aquí presente es responsabilidad de cada quién.

Las prácticas siguen la estructura solicitada de separar la lógica de la interfaz de usuario.

En cada practica en la clase de la gui `Pantalla` incluye el main y desde ahí se invocan las clases pertenecientes a la lógica.

Para visualizar el código se ingresa a la ruta:
   ```bash
   practica_x\src
   ```
Y se selecciona gui o logica según corresponda.
